File Server
-----------
A simple and temporary file hosting server.

How to use?
-----------
Open terminal and run the following command
    curl -F'file=@myfile' http://mysite.web/upload

curl will create a POST request to the server, along with the file which is in form body. Requests methods except POST will be rejected.

Maxium file size: 40 MiB

Rules (set your own rules as you wish)
----------------
You can upload basically everything you want, except these which are mentioned below :)
    * No malware
    * No NSFW/NSFL content
    * No homophobic
    * No pedo acts
    * Anything illegal
