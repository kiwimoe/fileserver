File Server
-----------
A small and simple file hosting server.

How to use?
-----------
Open terminal and run the following command
    curl -F'file=@myfile' http://mysite.web/upload

It will issue a POST request to the server, along with the file which is form body. Requests methods except POST will be rejected.

Maxium file size: 10 MiB
For how long files will stay here, vary on the file size. (See below)
